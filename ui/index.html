    🔬 Seul modèle fine-tunable du projet — Foundation-Sec-8B Instruct (Cisco)
    
            
    
              LLaMA 3.1 8B entraîné en continu par Cisco sur un corpus cybersécurité curé (CVEs, CTI, exploit write-ups, guides de conformité). Apache 2.0 — ~4.9 Go.
    
              C'est le seul modèle du projet basé sur une architecture ouverte non-propriétaire pouvant être fine-tuné via Unsloth QLoRA avec le dataset d'analyses du projet.
    
              Les autres modèles (Qwen 14B, Primus 8B) sont quantifiés ou propriétaires — non entraînables directement.
